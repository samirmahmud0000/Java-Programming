SecureGuard is a Java-based security validation framework that performs
multiple program-level security checks such as password validation,
SQL injection detection, XSS protection, token security, and data integrity.

## Features
- Password strength validation
- SQL Injection prevention
- XSS detection
- Token generation & validation
- Hash-based data integrity
- Clean and modular architecture

## Technologies
- Java
- Regex
- SHA-256
- UUID

## Use Case
Perfect for:
- Secure backend validation
- Academic projects
- Portfolio & GitHub showcase
